**1. What is the external fragmentation problem?** Exfrag happens when there's more than enough memory to fulfill the needs of the process queue, but those processes are allocated small, non-contiguous blocks that aren't able to fulfill the needs of other processes in the queue.


**2. Where is the base address of a segment stored?** The base address of a segment is stored in a descriptor table.


**3. What is the free list for segmented memory?** The free list is data structure that keeps track of unallocated physical memory.


**4. What is a thread?** A thread is the smallest executable chunk of a process.


**5. What is stored in a thread control block?** A data structure in the operating system that manages individual threads.


**6. What is a race condition?** Put simply, a race condition is when two processes executing concurrently try to write to the same program at the same time.


**7. What is a critical section of code?** A critical section of code is something that cannot be accessed concurrently, usually some kind of resource.


**8. Define atomic operation.** An atomic operation is a process that is executed completely and indivisibly on its own.


**9. What is a mutual exclusion?** A mutex is a property that prevents critical sections of code from being accessed concurrently.
