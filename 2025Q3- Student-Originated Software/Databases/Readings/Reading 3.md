In the Operating Systems thread, we learned about how the operating system interacts with memory quite a lot; if anything, it's the most important part of the course. Here, we learn about the different types of memory in and of itself. Learning about the different form factors of memory is critical in database design, since different forms use different fundamental methods of organization, which can alter how a database is written to it, how fast it can be processed, and the like.
