Reading 4 (Zybook 22)


It's MySQL time. This is where things got completely unfamiliar for me. We learn about MySQL's architecture, and the four layers of MySQL components: tools, which users themselves use to interface; the query processor, which takes multiple queries from multiple users and organizes them into instructions; the storage engine, which actually executes instructions; and the file system, which accesses data on whatever storage device is being used. The rest of chapter dives into the particularities of each of these in depth.
